# AI Resume Analyzer

AI-powered Resume Analyzer built using Python, Gradio, and PyPDF2.

## Features
- Resume PDF upload
- Skill extraction
- Resume statistics
- Job role recommendation

## Technologies Used
- Python
- Gradio
- PyPDF2

## How It Works
Upload a resume PDF and the application analyzes:
- skills
- candidate level
- suggested role
- resume statistics

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```
